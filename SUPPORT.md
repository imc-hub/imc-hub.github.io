## Reporting Bugs

If you encounter a bug on the website:

1. Check the [existing issues](https://github.com/imc-hub/imc-hub.github.io/issues) to see if it has already been reported
2. If not, open a new issue with:
   - A clear description of the problem
   - Steps to reproduce
   - Expected vs. actual behavior
   - Your browser and operating system

---

## Feature Requests

We welcome suggestions for new features or improvements. Open an issue with the tag `enhancement` describing:

- The feature you'd like to see
- The problem it solves or the value it adds
- Any relevant context or examples
